# AI工业化 - AI视频生成平台

仿制即梦AI (jimeng.jianying.com) 的AI视频创作平台。

## 功能

- 文生视频 (Text-to-Video)
- 图生视频 (Image-to-Video)  
- 首尾帧控制 (First Frame → Last Frame)
- AI绘画 (Text-to-Image)
- 智能画布 (Canvas Editor)
- 创意社区

## 技术栈

纯静态HTML + CSS + JavaScript，部署到 GitHub Pages。

## 使用

1. 配置 API Key
2. 打开 `index.html`

## 部署

```bash
git add .
git commit -m "init"
git push
# 启用 GitHub Pages 即可
```
